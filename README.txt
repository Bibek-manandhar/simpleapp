How to Run Build Script:

1. Ensure you have Python installed on your system.

2. Install setuptools by running the following command:

"pip install setuptools"


3. Clone the repository from GitHub:

git clone https://github.com/Bibekmanandar/simpleapp.git

4. Navigate to the project directory:

"cd simpleapp"

5. Run the build script:

"python setup.py install"

6. This will compile and test your application, as well as create a deployable package.

7. To run your application, use the following command:

my_application

8. You can find the executable or deployable package in the `dist` directory.
